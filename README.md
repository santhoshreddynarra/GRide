# 🚀 GigRide – Real-Time Gig Booking Platform

GigRide is a **full-stack MERN application** that enables real-time, on-demand booking of short-term jobs. Inspired by ride-hailing platforms, it connects **job providers** with **gig workers** instantly, making temporary hiring fast, efficient, and seamless.

---

## 🌟 Key Highlights

- ⚡ Real-time gig posting & acceptance
- 🔐 Secure authentication using JWT
- 👥 Role-based system (Provider / Worker)
- 📍 Location-based job opportunities
- 💰 Dynamic payment calculation
- ⭐ Rating & feedback system
- 📊 User dashboard for tracking gigs
